### Prerequisites

Install raylib (v4.5+ recommended):

**Ubuntu/Debian:**
```bash
sudo apt install libraylib-dev
# OR build from source:
git clone https://github.com/raysan5/raylib.git
cd raylib/src && make PLATFORM=PLATFORM_DESKTOP
sudo make install
```

**Arch Linux:**
```bash
sudo pacman -S raylib
```

```bash
make linux
./duel
```
